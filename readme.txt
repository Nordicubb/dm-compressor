Nordicub's NES-Style DPCM Compressor is a command-line program meant to turn audio files into 1-bit delta-modulated 7-bit PCM.

You need to install FFmpeg for this program to work.
https://www.ffmpeg.org/download.html

Nordicub's NES-Style DPCM Compressor v1.0.0 USAGE:
dpcmcomp [Input audio filename with extension] [output wav filename]
dpcmcomp [Input audio filename with extension] -- Output filename is output.wav.
dpcmcomp -- Opens Usage.
